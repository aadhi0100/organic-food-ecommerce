# 🌿 Organic Food E-Commerce

Premium organic food marketplace with advanced features.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Visit: http://localhost:3000

## ✅ Features

- Dark Mode Toggle
- Real-Time Notifications
- Advanced Search & Filters
- PWA Support (Install as App)
- Wishlist with Animations
- Social Sharing
- Smooth Animations
- Offline Support
